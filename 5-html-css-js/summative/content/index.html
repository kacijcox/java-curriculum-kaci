learn to trade crypto on solana

  1. download phantom wallet
  2. set up wallet
  3. fund your wallet with solana from binance, kraken, or moonpay
  4. visit dexscreener in phantom's built in browser to search for tokens
  5. next, visit the swap feature in phantom and paste the contract address
  6. be sure auto slippage is off to avoid mev bots
  7. happy trading!
